== Description == 
This plugin adds a download link below every image in your WordPress galleries.


== Gutenberg == 
This plugin currently only affects the old-school <a href="https://codex.wordpress.org/Gallery_Shortcode">gallery shortcode</a>. The gallery block, introduced with Gutenberg in WordPress 5.0, is not affected.


== Installation == 
1) Enable the plugin.
2) Add a gallery to your page/post with the gallery shortcode.
3) All images in the gallery will now have a link below them that says "Download".
